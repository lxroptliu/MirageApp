# MirageApp
MirageApp---回归通讯的本质
“在被‘3A 大作’式社交软件统治的荒漠中，我们决定亲手构建一片纯净的幻景。”
我们是谁？
我们是 Rust Mirage 工作室。我们是一群对内存安全有着近乎偏执追求的开发者，也是对个人数字主权绝不妥协的极客。我们厌倦了那些吞噬用户生活的“超级 App”。

核心准则
1. 软件应当各司其职
MirageApp 只用于聊天与通话。我们拒绝不相关的功能，专注于让沟通更快、更安全。

2. Rust：不仅仅是语言
极致轻量：安装包保持在 MB 级别。
丝滑响应：挑战毫秒级的消息递达延迟。
3. 隐私即人权
零知识架构：我们不持有密钥，无法读取你的数据。
数据主权：支持 WebDAV、个人邮箱或自建节点。
我们的愿景
我们创造一个“不妥协的工具”。MirageApp 不留住你的时长，只提供最纯净的通道。
